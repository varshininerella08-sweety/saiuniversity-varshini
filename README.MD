SAI UNIVERSITY WEBSITE 🎓

A responsive university website built using HTML, CSS, and JavaScript to showcase SAI University’s courses, campus, and facilities.

Features

- Home Page
- About University
- Courses Section
- Admissions Information
- Contact Page
- Responsive Design
- Modern UI

How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Run "index.html" using Live Server.
